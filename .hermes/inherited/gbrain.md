# GBrain (real rule)

Authoritative source: https://github.com/garrytan/gbrain (Garry Tan, YC).
Evals sibling: https://github.com/garrytan/gbrain-evals.

## What it actually is

A brain / retrieval layer for AI coding agents (Claude Code / Codex): a synthesis
layer that returns a cited answer plus an explicit gap analysis (what the brain
does NOT know yet) over a self-wiring knowledge graph — not "here are 10 chunks
that mention your query". The discipline: synthesize sourced answers AND name the
gaps; never dump raw chunks.
