# Garde-fous agents Sircom 2026

Ce dossier contient les prompts et checklists de pilotage des agents
d'implémentation.

Règle principale : un agent travaille sur un seul ticket à la fois. Le statut
`ready-for-agent` signifie que le ticket est cadré, pas que les tickets aval
peuvent être anticipés.
